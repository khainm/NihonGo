# Luồng UI Luyện Tập Kanji

```
HomePage
    ↓ (Chọn cấp độ N5)
LevelDetailPage 
    ↓ (Chọn "Kanji")
KanjiLessonsPage
    ↓ (Chọn "Bài 1")
KanjiLessonDetailPage
    ↓ (Nhấn "Bắt đầu luyện tập")
KanjiPracticePage
    ├── Menu chọn phương thức (Trang chính)
    ├── Flashcard (Lật thẻ học)
    ├── Trắc nghiệm (Câu hỏi)
    ├── Viết nét (Hướng dẫn)
    └── Thống kê (Tiến độ)
```
